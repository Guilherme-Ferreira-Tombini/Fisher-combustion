# Fisher Combustion

Está aplicação tem como objetivo o calculo de consumo de veiculos e seus gastos em dinheiro com combustivel.


## Funcionalidades

- Temas dark e light;
- Calculos de consumo;
- Geração de PDF;
- Multiplataforma.
